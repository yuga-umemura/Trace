# 男性向け服の買い物支援アプリ

## 1. プロダクト概要

### コンセプト

20,30代男性が自分に合うお店の選択肢を広げる

### 解決する課題

- 自分に合う店がわからない
- 店に行っても「これがいい」と思える服が見つからないことがある
- 欲しい服が見つからなかったときに、次にどこへ行けばいいかわからない

本質的な課題は、「自分に合うお店の探し方」がわからないこと

### コアな価値

多くの選択肢の中から、自分が“いい”と思える服に出会える店選びを支援し、見つからなかった場合でも次の選択肢を提示する

### 解決アプローチ

- スタイル診断でユーザーのタイプを判定する
- タイプごとに合う店舗をレコメンドする
- 店に行っても見つからなかった場合に、次の店舗を提案する
- 「このタイプの人は〜」という形で、自分に合う選択肢を広げる
